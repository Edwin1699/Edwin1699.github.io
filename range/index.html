Range

Introducing my sans-serif typeface, Dyspo. This typeface is meant to give a futuristic vibe with gaps in each letter. I also aimed to create symbols for each letter so it isn't just a letter that looks futuristic. At first, it was very geometric, but I felt like the square letters created a limited sense in design. So in turn I used cirlces and curves on some spots to keep the typeface a bit interesting in the long run.
